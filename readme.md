# CSS Selectors Practice

This project demonstrates the use of basic HTML and CSS selectors.

# Features

- Linked an external CSS file (`style.css`) to `index.html`
- Set a background color for the webpage
- Styled multiple headings using a class selector
- Styled a specific heading using an ID selector
- Styled paragraphs using the element selector
- Customized a specific paragraph using an ID selector
- Styled a single word using a nested selector
- Styled links without using inline CSS


# Files

- `index.html` – Contains the HTML structure.
- `style.css` – Contains all the CSS styling.
- `README.md` – Project documentation.

# Technologies Used

- HTML5
- CSS3

# How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in any web browser.

# Learning Objectives

This project practices:
- Element selectors
- Class selectors
- ID selectors
- Descendant selectors
- Pseudo-class selectors
- External CSS
